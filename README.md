# rms\_contrast

* JPEG/TIFF형식 이미지 파일의 컨트라스 값을 계산.
* 촬영 이미지의 컨트라스트 값이 가장 높아지도록 AF 미세조정 작업을 수행.



미러리스나 라이브뷰에서 사용되는 컨트라스트 AF는

같은 조건에서 이미지의 대비값(컨트라스트)이 커질 수록 핀이 맞다는 사실을 이용해서 

컨트라스트가 최대가 되는 지점을 찾아서 초점을 맞추는 방식으로 작동합니다.



같은 원리를 이용해서 AF 미세조정값을 바꿔가면서 촬영한 결과물의 대비값(컨트라스트)을 비교해서

컨트라스트가 최대가 되는 지점을 찾아서 AF 미세조정(핀 교정)을 할 수 있습니다. 



이 프로그램은 D~Z에 있는 모든 드라이브에서 DCIM 디렉토리를 찾아서

그 안에 있는 모든 JPEG, TIFF파일의 컨트라스트를 계산한 후,

엑셀(이나 다른 스프레드시트)에서 불러들일 수 있는 CSV형식으로 결과값을 저장해 줍니다.

저장된 값을 스프레드시트에서 불러들인 후, 차트를 생성하면 컨트라스트가 최대가 되는 지점을 쉽게 찾을 수 있습니다.



시작하기 전에 세팅을 정확하게 해야 좋은 결과를 얻을 수 있습니다. 


1. 모니터와 카메라 센서가 평행이 되도록 정렬.

2\. AF 미세조정값 이외에는 다른 변수가 없도록 고정.





1\. 모니터와 카메라 센서가 평행이 되도록 정렬.

&nbsp;	- 테스트용 동심원 이미지를 화면에 띄웁니다.

&nbsp;	- CD 혹은 거울 역할을 할 수 있는 반사판을 이미지 중심부에 테이프로 붙입니다.

&nbsp;	- 카메라를 삼각대에 거치한 후, 모니터에 장착된 거울(혹은 거울 역할을 하는 반사판)을 향했을 때, 뷰파인더나 라이브뷰 화면의 중앙에 카메라 본체와 렌즈가 보이면 됩니다.

&nbsp;	- 이렇게 모니터와 카메라 센서가 평행으로 정렬되면, 모니터의 모든 영역이 센서와의 거리가 균일하게 됩니다. 

&nbsp;	- 카메라 위치가 바뀔 때마다 다시 정렬을 해 주고, 정렬을 마치면 거울 역할을 하는 반사판을 떼어냅니다.



2\. AF 미세조정값 이외에는 다른 변수가 없도록 고정.

&nbsp;	- M(수동)모드를 사용해서 조리개값, 셔터속도를 고정합니다.

&nbsp;	- ISO를 자동(AUTO)이 아닌 값으로 지정해서 고정합니다.

&nbsp;	- 화이트 밸런스(WB)를 자동(AWB)이 아닌 값으로 지정하여 고정합니다.

&nbsp;	- 설정 메뉴에서 노이즈 제거, 렌즈 보정, 노출 보정, 손떨림 방지등 이미지에 영향을 미칠 수 있는 모든 기능을 끕니다.

&nbsp;	- 커스텀 이미지 설정에서 '샤픈' 값을 0으로 설정합니다.

&nbsp;		(초점이 안 맞으면 이미지가 흐릿해져서 컨트라스트 값이 낮아지는 원리를 이용해서 초점이 맞는 지점을 찾는 데,

&nbsp;		바디에서 샤픈 보정해 버리면 이미지가 흐릿해지는 게 줄어들어서 초점 맞는 지점을 찾기가 힘들어 집니다.) 

&nbsp;	- 촬영을 할 때 셀프 타이머, 셔터 릴리즈, 인터벌 촬영등의 방법을 사용해서 셔터 충격을 최소화하도록 합니다.

&nbsp;	- 테스트 하는 동안에 카메라가 움직이지 않도록 삼각대에 단단히 고정합니다. 

&nbsp;	- 형광등 불빛은 빠른 속도로 깜빡이므로 테스트에 영향을 줄 가능성이 있습니다. 자연광이나 LED 조명등이 바람직합니다.



3\. 테스트 이미지 촬영 

&nbsp;	- 촬영을 시작하기 전에 메모리 내용을 비워줍니다. (나중에 PC에서 컨트라스트 값을 계산할 때 불필요한 계산을 하지 않아서 시간이 단축됩니다.)

&nbsp;	- 촬영 과정 중에 발생하는 진동을 최소화 하십시오.

&nbsp;	- 손으로 셔터를 누르는 것 : 최악입니다.

&nbsp;	- 리모콘을 사용하거나 인터벌 촬영 기능을 사용하는 것 : 차선입니다.

&nbsp;	- 리모콘 + 미러업 촬영 기능을 촬영하는 것 : 최선입니다. 상당히 깨끗한 그래프를 얻을 수 있습니다.

&nbsp;	- 진동이 발생할 수록 오차가 심해지므로 촬영횟수를 늘려서 눈으로 보고 너무 튀는 값은 제거하거나, 평균값에서 오차로 인한 영향을 줄이는 식으로 대응합니다. 

&nbsp;	- A미세조정값을 최소값에서부터 최대값까지 1씩 증가시키며, '미러업 + 리모콘' 촬영의 경우 각 3회씩, 그렇지 않은 경우에는 각 5회씩 촬영합니다. 

&nbsp;	- AF 미세 조정할 렌즈 수량이 많은 분들은 추가 배터리를 준비해 둡니다.



4\. 촬영한 이미지 분석

&nbsp;	- 촬영을 마친 후에 PC의 리더기에 메모리 카드를 꽂은 후,  컨트라스트 값을 계산하는 프로그램을 실행시킵니다.

&nbsp;	- 계산이 끝나면 결과값이 실행화일이 위치한 디렉토리에  '대비값\_<날짜>-<시간>.csv' 파일로 저장됩니다.

&nbsp;	- 엑셀(혹은 여타 스프레드시트)에서 csv 파일을 불러들입니다.

&nbsp;	- 첫번째 열은 파일명이고, 두번째 열은 컨트라스트 값입니다.

&nbsp;	- 파일명은 어느 렌즈에서 촬영한 이미지인지 구분하기 위한 목적일 뿐, 실제 분석에는 필요없습니다.

&nbsp;	- 두번째 열에 대해서 막대 그래프를 생성합니다.

&nbsp;	- 눈으로 그래프를 본 후 유독 낮게 나오는 값은 제거해 줍니다.   (촬영할 때 흔들리거나, AF 백래쉬 오차 때문입니다. 각 5회 촬영했기 때문에 1~2개 지워도 별 지장 없습니다.)

&nbsp;	- 그래프가 산 모양으로 나오면 테스트가 제대로 수행된 것이며, 그래프의 정상 지점에서 AF미세조정값이 핀이 맞는 지점입니다.

&nbsp;	- 만약, 어느 지점이 정상인지 구분하기 힘들 경우에는 스프레드시트의 기능을 이용해서 평균값을 구한 후 비교하면 됩니다.

&nbsp;	- 평균값끼리도 비슷비슷해서 구분이 가지 않으면, 비슷한 값만 추려서 새로이 그래프를 생성하면, Y축의 범위가 조정되면서 차이가 확대되어서 구분이 됩니다.

&nbsp;	- 엑셀이나 여타 스프레드시트 프로그램이 없으면, 무료 프로그램인 '오픈 오피스', '리브레 오피스'도 간단하게 쓸만 합니다.



<추가 1>

v0.0.2부터 TIFF형식을 지원합니다.

JPEG형식은 압축과정에서 손실이 있지만, TIFF형식은 무손실 방식입니다.

K-1처럼 바디 내에서 TIFF변환이 가능한 경우에는, JPEG보다 RAW촬영 후 TIFF로 바디 내 전개로 생성하면 더 정확한 결과를 얻을 수 있을 듯 합니다.

조리개값이 밝은 단렌즈의 경우 조리개를 약간 조인 후 테스트 해 보시기 바랍니다.

F1.4처럼 심도가 극도로 얕은 경우에는  AF 백래쉬 오차, 미러 충격등 다른 변수에 컨트라스트 값이 팍팍 튀어버려서 차트가 굉장히 불규칙해 집니다.

이럴 경우, F4 정도로 조리개를 조여주니까 그래프 튀는 게 거의 사라지고, 안정적인 차트 생성이 가능했습니다.
